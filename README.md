# 🚀 Como executar o projeto

Clone o repositório:

```bash
git clone https://github.com/IssamuAbe/portfolio-issamu.git
```

Entre na pasta:

```bash
cd portfolio-issamu
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```

A aplicação estará disponível em:

```text
http://localhost:5173
```

---

# 🏗 Build de produção

Gerar a build:

```bash
npm run build
```

Visualizar a build localmente:

```bash
npm run preview
```
